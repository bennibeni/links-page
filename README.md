# Links page

Piccola homepage per raccogliere i link pubblici dei progetti Vercel.

## Sviluppo locale

```bash
npm install
npm run dev
```

Poi apri `http://localhost:3000`.

## Aggiornare i link

Modifica l'array `projects` in `app/page.js`.

## Deploy su Vercel

1. Crea un nuovo progetto Vercel puntando alla cartella `links-page`.
2. Come nome progetto puoi usare `i-miei-progetti`.
3. Vercel generera un URL simile a `https://i-miei-progetti.vercel.app`.
